# itp405-spring2015-mvc

### The DVD App
#### DVDs por favor :)

### App Features
+ view all dvds
+ search for dvds
+ view all genres
+ view all dvds in a genre
+ write reviews of dvds
+ view reviews of dvds
+ create dvds
+ view Rotten Tomato data

### Technologies Used
+ Laravel / MVC
+ PHP / HTML / CSS
+ Bootstrap CSS
+ HTML forms
+ MySQL database
+ Blade templating
+ Eloquent ORM
+ Eager loading
+ Rotten Tomatoes API
+ Laravel's Cache class
